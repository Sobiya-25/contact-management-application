# Contact Management System

A web-based Contact Management System built using Flask and SQLite to manage user contacts efficiently.

## 🚀 Features
- Add, update, delete, and view contacts (CRUD operations)
- Server-side validation for email and mobile number
- Duplicate prevention using unique constraints
- Responsive UI using Bootstrap

## 🛠️ Tech Stack
- Backend: Flask (Python)
- Database: SQLite
- Frontend: HTML, CSS, Bootstrap
